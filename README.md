# Customer Churn Prediction

## Project Overview
This project aims to predict customer churn using machine learning models.

## Dataset
IBM Telco Customer Churn Dataset

## Project Goals
- Exploratory Data Analysis (EDA)
- Data Preprocessing
- Model Building
- Model Evaluation
- Hyperparameter Tuning

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- Gradient Boosting

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC
